# CSS

**CSS stands for Cascading Style Sheets , and it is a styling language used to define the appearance and layout of web pages written in HTML or XML.**

## Inline CSS

```html
<style>
  properties
</style>
```

## External CSS

```html
<link rel="stylesheet" href="./style.css" />
```
